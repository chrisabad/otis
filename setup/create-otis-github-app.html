Create Otis GitHub App

One-click registration for the Otis AGE agent GitHub App. GitHub will pre-fill the app settings from this manifest.

Permissions this app will request

  • Contents — read & write (commit files, push branches)
  • Pull requests — read & write (open and update PRs)
  • Issues — read & write (create and comment on issues)
  • Workflows — read & write (update GitHub Actions files)
  • Metadata — read (required by GitHub)
After GitHub creates the app:
  1. Note the App ID shown on the app settings page
  2. Scroll down and click Generate a private key — save the .pem file
  3. Click Install App → install on chrisabad/otis and any other repos Otis needs
  4. Note the Installation ID from the install URL (/installations/<id>)
  5. Store App ID, Installation ID, and private key in AWS Secrets Manager as agentos/otis/github_app
